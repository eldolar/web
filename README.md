#### Develop
Install dependencies
* ```$ yarn install```

Lunch server
* ```$ yarn start```

Run test in watch mode
* ```$ yarn test```

Run StoryBook
* ```$ yarn storybook```

Build
* ```$ yarn build```

Deploy to Gh-Pages
* ```$ yarn deploy```
